# Markdown 扩展示例

## 自定义容器

```js
::: tip 提醒
这里是 tip 容器
:::

::: warning 警告
这里是警告容器
:::

::: danger 危险
这里是危险容器
:::
```

::: tip 提醒

:::

::: warning 警告

:::

:::

## 链接

### 锚点链接

在 VuePress 中所有.md 文件中的标题(默认 h2 和 h3)都会自动添加锚点链接(anchor)，所以如果我们需要跳转至固定的锚点，可以如下进行设置

```js
[锚点链接](/example/#自定义容器)
```
